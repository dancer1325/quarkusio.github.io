* Observability
    ** requires orchestrate & connect > 1 Dev Service
        *** _Example:_
            **** metrics agent periodically scrape application for metrics / push them | timeseries database
            **** Grafana feeds graphs of this timeseries data

* Dev Resource
    ** == adapter between Dev Service concept & https://testcontainers.com/[Testcontainers]
    ** 's implementation == `@QuarkusTestResourceLifecycleManager` implementation
    ** ways to use
        *** Dev Services can pick-up various Dev Resources | classpath + apply xref:dev-services.adoc[Dev Service] concept
        *** explicitly disable Dev Services + enable Dev Resources + use starting and stopping Dev Resources' less-heavy concept
        *** explicitly disable Dev Services & Dev Resources + use Quarkus' `@QuarkusTestResource`

* ways
    ** add Observability extension dependency + Dev Resources dependencies, OR
    ** use existing `sinks`
        *** == pom.xml files / add
            **** Observability extension dependency
            **** other required dependencies / certain technology stacks
        *** _Example:_ `victoriametrics` sink == `quarkus-observability-devresource-victoriametrics` + `quarkus-victoriametrics-client` dependencies /
